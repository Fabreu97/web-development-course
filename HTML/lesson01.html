HTML é a linguagem de marcação padrão para páginas da Web que é acrônimo de Hype Text Markup Lanaguage(Linguagem de Marcação de Hipertexto).

Estruturas de uma página Web

  • head
  • body

Head

Fica as informações de cabeçalho e configurações da página.Container onde fica os metadados

Body

É o lugar onde fica o conteúdo da página.
